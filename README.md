# calculatrice

## Durée :
7h

## Ennoncé
En utilisant **Tkinter**, votre mission sera de réaliser une **calculatrice**.

La contrainte à ce projet, est que la calculatrice ne possède qu’au **maximum trois boutons** :
- Bouton 'C' : effacer
- Bouton '+' : additionner
- Bouton '=' : calculer
